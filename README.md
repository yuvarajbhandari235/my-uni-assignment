# Yuvaraj Bhandari Employability Skills Portfolio

A responsive single-page HTML, CSS, and JavaScript website created for Assessment Task 3 of BSM24204 Entrepreneurism.

## Files

- `index.html`: Website content and semantic structure
- `styles.css`: Responsive visual design and animation
- `script.js`: Navigation, reveal animation, scroll highlighting, and image lightbox
- `assets/images`: Optimised evidence images
- `assets/video`: Event evidence video

## Preview locally

Open `index.html` in a modern browser. For the most reliable preview, use a local server:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Publish online

The complete folder can be uploaded to GitHub Pages, Netlify, or another static website host. Keep the folder structure unchanged.
